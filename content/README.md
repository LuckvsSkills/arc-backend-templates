# ARC Content Backend Template

## Wat zit erin
- Artikelen met versioning en planningssysteem
- Pagina's met templates
- Media bibliotheek
- Commentaren met moderatie
- Nieuwsbrief systeem
- SEO instellingen
- Menu beheer
- RSS feed generatie
- Docker deployment ready

## API Endpoints

| Module | Basis URL |
|--------|-----------|
| Authenticatie | /api/v1/auth |
| Artikelen | /api/v1/artikelen |
| Categorieën | /api/v1/categorieen |
| Pagina's | /api/v1/paginas |
| Media | /api/v1/media |
| Nieuwsbrief | /api/v1/nieuwsbrief |
| SEO | /api/v1/seo |
| Admin | /api/v1/admin |

## Compatibel met
Frontend: blog-v1/2/3/4, portfolio-v1/2/3/4, landing-v1/2/3/4
