# ARC Marketplace Backend Template

## Wat zit erin
- Multi-vendor platform met verkoper profielen
- Advertentiebeheer met moderatie
- Transactiesysteem met commissies
- Automatische uitbetalingen aan verkopers
- Berichten tussen kopers en verkopers
- Beoordeling en review systeem
- Melding en moderatie systeem
- Volledige tekst zoekfunctie
- Docker deployment ready

## API Endpoints

| Module | Basis URL |
|--------|-----------|
| Authenticatie | /api/v1/auth |
| Verkopers | /api/v1/verkopers |
| Advertenties | /api/v1/advertenties |
| Transacties | /api/v1/transacties |
| Berichten | /api/v1/berichten |
| Beoordelingen | /api/v1/beoordelingen |
| Zoeken | /api/v1/zoeken |
| Admin | /api/v1/admin |

## Compatibel met
Frontend: marketplace-v1/2/3/4, directory-v1/2/3/4
Admin: admin-marketplace
Agent: agent-marketplace
