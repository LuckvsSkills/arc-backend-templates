# ARC Community Backend Template

## Wat zit erin
- Gebruikersprofielen met volgen systeem
- Groepen en kanalen (openbaar/besloten/privé)
- Posts met reacties, likes en emoji reacties
- Privéberichten en groepsgesprekken
- Evenementen met aanmelding
- Badges en gamification systeem
- Moderatie en meldingen
- Real-time notificaties via WebSocket
- Docker deployment ready

## API Endpoints

| Module | Basis URL |
|--------|-----------|
| Authenticatie | /api/v1/auth |
| Gebruikers | /api/v1/gebruikers |
| Groepen | /api/v1/groepen |
| Posts | /api/v1/posts |
| Berichten | /api/v1/berichten |
| Evenementen | /api/v1/evenementen |
| Badges | /api/v1/badges |
| Admin | /api/v1/admin |

## Compatibel met
Frontend: community-v1/2/3/4, saas-v1/2/3/4
Admin: admin-community
Agent: agent-community
